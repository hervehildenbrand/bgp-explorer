# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [0.1.0] - 2025-01-19

### Added
- Initial release
- AI-powered BGP routing investigation CLI
- Integration with RIPE Stat API for BGP data
- Real-time monitoring via bgp-radar
- Globalping integration for network probing
- PeeringDB integration for IXP information
- BGPKIT Monocle integration for AS relationships
- Claude AI backend for natural language queries
- Rich CLI output with monitoring badges
- Tab completion for commands
